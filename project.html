1. Project Title: SaturnEdge Website
Project Description:
The SaturnEdge Website is a live project developed using HTML and CSS to showcase a collection of products. This project is designed to provide an interactive and visually appealing platform for users to browse and explore various products.
Visit: saturnEdge

Key Features:
  • Product Display: The website displays a grid or list of products, each image represented as moon of saturn. Users can easily see the products available for sale.
  • Responsive Design:The project is responsive, ensuring a consistent and enjoyable experience on various devices, including desktops, tablets, and smartphones.
  • CSS Transitions and Animations: CSS is used to add subtle transitions and animations, enhancing the visual appeal and user interactivity.

Technologies Used:
  • HTML5: For structuring the web pages.
  • CSS3: For styling and layout.
  • JavaScript (optional): For adding interactivity and dynamic features.

Challenges:
  • Designing an attractive and intuitive user interface.
  • Implementing responsive design for various screen sizes.

3. Project Title: Contact Manager
Project Description:
The Contact Manager App is a web-based application built using React, a popular JavaScript library for building user interfaces. This app serves as a digital address book, enabling users to manage their contacts efficiently. It provides a user-friendly interface for adding, viewing, and deleting contacts.
Visit: Contact Manager

Key Features:
  • Contact List: Display a list of contacts with their names and thumbnail images for easy identification.
  • Add New Contacts: Users can add new contacts to their address book by filling out a form that includes fields like name, email, phone number, and more.
  • Delete Contacts: Enable users to delete contacts they no longer need.

Technologies Used:
  • React: For building the user interface and handling the application's state.

Challenges:
  • State Management: Managing the state of contacts and ensuring data consistency.
